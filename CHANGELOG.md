# Changelog

All notable changes to this project are documented here. The format is based
on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project
adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Changed (per Greg Gleason / EOG review, 2026-05-07)
- **VNF attribution corrected.** Removed all "NOAA's VIIRS Nightfire"
  framing. VNF is produced by the Earth Observation Group (EOG) at the
  Payne Institute, Colorado School of Mines — NOAA only operates the
  VIIRS instrument on the Suomi NPP and JPSS-1 satellites. README,
  module docstrings, all four country notebooks, frontend prose, docs,
  and Y4 outreach drafts updated.
- **`fetch_ggfr_annual()` → `fetch_gfmr_annual()`** to match the World
  Bank's 2025 rename of the Global Gas Flaring Reduction Partnership
  (GGFR) → Global Flaring and Methane Reduction Partnership (GFMR).
  `fetch_ggfr_annual` and `GGFR_DATA_URL` remain as deprecated aliases
  for v0.1 / v0.2; will be removed in v0.3. Same for the private helpers
  `_melt_ggfr_wide` / `_melt_gfmr_wide` and
  `_standardize_ggfr_schema` / `_standardize_gfmr_schema`.
- **License §1.b posture restated up front.** README, LICENSING_NOTES,
  module docstrings, `fetch_vnf_nightly()` docstring all now lead with
  "pyflare does not bundle, redistribute, mirror, or rehost VNF data" —
  the library is a thin client; each user must hold their own VNF
  Academic Data Use License.

### Changed
- **PyPI distribution renamed `pyflare` → `pyflare-africa`.** The original
  `pyflare` name on PyPI is occupied by an abandoned 2015 Cloudflare API
  adapter. The Python import name is unchanged (`import pyflare as pf`)
  — only the install command changes (`pip install pyflare-africa`).
  README, badges, and docs updated to reflect this.

### Added (Day 1 closeouts)
- `docs/` — Sphinx + Furo documentation scaffold with auto-API
  generation from NumPy docstrings (extensions: napoleon, viewcode,
  intersphinx, autodoc-typehints, myst-parser). Pages: index,
  quickstart, api, licensing. Builds locally with
  ``sphinx-build -b html docs docs/_build``.
- `.readthedocs.yaml` — Read the Docs config (Ubuntu 22.04, Python 3.12,
  installs the `docs` extras).
- `RELEASE.md` — end-to-end release runbook covering pre-release
  checks, clean wheel build, smoke install in a fresh venv, TestPyPI
  dry run, production upload, GitHub Release with assets, and
  common-failure fixes.
- `src/pyflare/dashboard.py` — Streamlit MVP exposing the GFMR trend,
  CO₂-equivalent slider, top-5 producer comparison, and
  communities-near-sites exposure for the four priority countries.
  Local-only by design — public deployment requires written EOG
  approval per VNF license §1.f.iii.
- `dashboard` and `docs` extras in `pyproject.toml`.
- `dist/pyflare_africa-0.1.0-py3-none-any.whl` and
  `dist/pyflare_africa-0.1.0.tar.gz` built; smoke-installed in a fresh
  venv to verify ``import pyflare`` works and ``volume_to_co2eq(6.48,
  0.05) == 22.62 MtCO₂e`` end-to-end.

### Changed (breaking — pre-1.0)
- `fetch_vnf_nightly()` now uses OAuth 2.0 (Keycloak password grant)
  instead of HTTP basic auth. EOG migrated the VNF API to bearer-token
  access in early 2026; basic auth no longer works against
  `eogdata.mines.edu`. New required env vars: `EOG_CLIENT_ID`,
  `EOG_CLIENT_SECRET` (issued by EOG on request once the academic
  license is signed), in addition to the existing `EOG_USERNAME` and
  `EOG_PASSWORD`. Tokens are cached in-process for ~5 minutes and never
  written to disk. New helper: `_get_eog_access_token()`.
- New module constant `EOG_TOKEN_URL` documents the Keycloak token
  endpoint.

### Added (poster assets — Day 1)
- `notebooks/02_angola.ipynb`, `notebooks/03_algeria.ipynb`,
  `notebooks/04_libya.ipynb` — sister notebooks to the Niger Delta
  walkthrough, generated by a country-parameterised builder
  (`scripts/_build_country_notebooks.py`). Each runs end-to-end with
  synthetic VNF, GFMR-published 2024 volume, and produces a
  CO₂-equivalent headline range.
- `assets/africa_overview_600dpi.png` — iconic Africa map at 600 dpi
  (3600 × 4200 px), produced by `scripts/_build_iconic_africa_map.py`.
  Dark theme, 17 producer countries highlighted, layered-alpha flare
  glow on the four priority countries' synthetic facility coordinates.
  Carries the §2.a credit line in the footer.
- `assets/africa_timelapse.gif` — 5-frame timelapse (2012, 2015, 2018,
  2021, 2024) showing African flaring intensity by year. Per-country
  GFMR volumes drive dot brightness, normalised against the African
  producer maximum across the dataset (9.62 bcm). Produced by
  `scripts/_build_iconic_timelapse.py` (1200 × 1350 px, ~134 KB).
- `pyflare.analysis.communities_near_sites()` — screening-level
  population-exposure analysis: returns nearby communities + summed
  population for each flare site. Default reference is the curated
  `KNOWN_AFRICAN_SETTLEMENTS` dict (Niger Delta over-represented; expand
  by country as country maintainers are recruited). Niger Delta
  synthetic 6-site exposure: ~322,000 within 10 km, 2.2 M within 25 km.
  7 unit tests covering the function.

### Added (licensing compliance)
- `LICENSING_NOTES.md` — developer-facing summary of the VNF Academic
  Data Use License obligations (no raw redistribution, attribution
  required, weekly minimum public temporal aggregation, year-N volume
  publication only after EOG publishes year-N, EOG/Mines endorsement
  framing, year-by-year disclosure clearance table).
- README acknowledgements expanded with the verbatim VNF attribution
  notice (license §2.a) and a link to `LICENSING_NOTES.md`.
- Notebook header includes a data-attribution cell that activates when
  the synthetic VNF block is replaced with `pf.fetch_vnf_nightly()`.

### Fixed
- `fetch_ggfr_annual()` now points at the 2025 World Bank GFMR XLSX release
  (the previous CSV URL returns 404). The fetch reads sheet ``Flare volume``
  and melts the wide format to a tidy ``country`` / ``year`` / ``bcm_flared``
  frame. World Bank country labels are normalised to pyflare's canonical
  set so the result composes with `filter_country()`.

### Added
- `pyflare.analysis.volume_to_co2eq()` — combustion + methane-slip
  CO2-equivalent estimator. IPCC 2006 (combustion EF 2.55 kg/m³) +
  IPCC AR6 (GWP-100 fossil CH4 = 29.8) + configurable slip. Returns
  megatonnes CO2-equivalent.
- New module constants: `CO2_PER_M3_GAS_KG`, `METHANE_GWP100_FOSSIL`,
  `METHANE_DENSITY_KG_PER_M3` (the last shared with `methane_proxy`).
- `notebooks/01_niger_delta.ipynb` — end-to-end Niger Delta walkthrough:
  GFMR trend → African context → synthetic VNF pipeline → CO2-eq
  headline. Runs without EOG credentials.
- New core dependency: `openpyxl>=3.1` (for the GFMR XLSX).

## [0.1.0] - 2026-05-16

### Added
- Initial public scaffold.
- `pyflare.data`: GFMR annual fetch, VNF nightly fetch, country and bbox
  filters, schema normalization for 17 African oil/gas-producing nations.
- `pyflare.analysis`: detection-type classification (Elvidge et al. 2013),
  persistence scoring, greedy single-link site clustering, flared-volume
  estimation, configurable methane proxy with literature-cited slip fraction.
- `pyflare.viz`: Folium maps, matplotlib trend charts, project palette,
  Africa-overview map.
- 38 unit tests; no network dependency in the suite.
- Contributor Covenant 2.1 code of conduct.
- CI matrix on Python 3.10 / 3.11 / 3.12 with ruff.
- Pre-commit config (ruff, mypy, standard hygiene hooks).
- Adopt-a-Country contributor programme.
